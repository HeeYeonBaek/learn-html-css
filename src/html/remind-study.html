단락 감싸는 방법

alt + W : 양 옆을 태그로 감싼 후 태그입력
이는 emmet 기능 중 하나로써 원래는 단축키가 따로 있지 않다.

  1. Ctrl+Shift+P를 눌러서 명령팔레트를 실행합니다.
  2. 명령팔레트 창에 emmet 라고 검색합니다.
  3. 검색 결과 중 emmet:Wrap With Abbreviation을 선택합니다.
  4. 이때 상단에 입력창이 나오면 원하는 태그를 입력하고 뒤에 *을 추가하고 엔터를 누릅니다. (예시 -> li별표)

위의 기능을 요약해서 사용한다면 "ul>li*갯수" 형태로 작성할 수 있을 것 같다.
